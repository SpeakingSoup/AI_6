# Лабораторная работа №6: "Решение задачи регрессии с помощью НС"

Данный репозиторий содержит файлы выполненной лабораторной работы №6

## Содержание

- *lite* – задание уровня Lite. Цель - провести ряд экспериментов с гиперпараметрами для определения лучших с наименьшей MAE.
- *lite_pytorch* – задание уровня Lite, выполненное с помощью библиотеки pytorch. 
- *pro* – задание уровня Pro. Цель - добиться ошибки меньше 17.
- *ultra_pro* – задание уровна Ultra Pro. Цель - создать модель, предсказывающую цены на машины.
